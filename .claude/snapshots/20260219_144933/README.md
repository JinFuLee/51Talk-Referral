# 51Talk 转介绍周报自动生成

自动处理转介绍业务数据，生成标准化运营分析Excel报告。

## 快速使用

```bash
# 处理最新数据文件
python3 main.py --latest

# 监控模式（自动处理新文件）
python3 main.py --watch

# 处理指定文件
python3 main.py --once /path/to/data.xlsx
```

## 智能时间进度

程序自动计算月度时间进度：

- **T-1 数据**: 今天处理的是昨天的数据
- **周末加权 1.4x**: 周六日业绩通常更高
- **周三降权 0.7x**: 周三通常较低
- **其他工作日**: 标准权重 1.0x

## 配置月度目标

编辑 `config.py`:

```python
MONTHLY_TARGETS = {
    "202601": {
        "注册目标": 779,
        "付费目标": 179,
        "金额目标": 147848,
    }
}
```

## 输出报告

| Sheet | 内容 |
| ----- | ---- |
| Sheet1 | 月度同期分析（环比公式、目标进度） |
| Sheet2 | 原始数据完整复制 |

---

*报告输出目录: `output/`*
