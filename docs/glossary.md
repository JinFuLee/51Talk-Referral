# 业务术语表 — 51Talk 泰国转介绍

## 组织架构

| 层级 | 说明 | 示例 |
|------|------|------|
| **国家** | 最高层级 | 泰国 |
| **部门** | CC/SS/LP 三大部门 | CC前端、SS后端、LP服务 |
| **Team** | 团队 | TH-CC01Team, TH-SS01Team, TH-LP01Team |
| **个人** | CC/SS/LP 个人 | THCC-patimakorn, THSS-thitima |

## 团队类型

| 代码 | 全称 | 角色 | 数据中的别名 |
|------|------|------|------------|
| **CC** | CC前端 | 前端销售，负责获客、跟进、促付费 | CC |
| **SS** | SS后端 | 后端销售，负责续费、增购 | **EA**（数据中常见） |
| **LP** | LP服务 | 后端服务，类似学生的老师 | **CM**（数据中常见） |

> **重要**: 面板/报告统一使用 **CC/SS/LP**，遇到 EA 自动映射为 SS，遇到 CM 自动映射为 LP。

## 渠道类型

| 术语 | 定义 | 质量 |
|------|------|------|
| **窄口（径）** | CC/SS/LP 绑定新学员 | 高（员工主动推荐） |
| **宽口（径）** | 用户（老学员）绑定新学员 | 低（学员自然分享） |

细分：
- **CC窄口径**: CC 员工链接绑定
- **SS窄口径**: SS(EA) 员工链接绑定
- **LP窄口径**: LP(CM) 员工链接绑定
- **宽口径**: 学员链接绑定（不区分部门）

## 围场

| 术语 | 定义 |
|------|------|
| **围场** | 用户**付费当日**开始计算的天数分段 |
| 0-30 天 | 付费后 0-30 天内的学员 |
| 31-60 天 | 付费后 31-60 天的学员 |
| 61-90 天 | 付费后 61-90 天的学员 |
| 91-180 天 | 付费后 91-180 天的学员 |
| 181+ 天 | 付费超过 181 天的学员 |
| 未付费 | 尚未付费的学员 |

## 关键 KPI 指标

### 学员层指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **有效学员** | 次卡 > 0 且在有效期内的海外学员 | 可参与转介绍的基础池 |
| **触达率** | 当月有效通话学员 / 当月有效学员 | 有效通话 = 通话时长 >= 120 秒 |
| **参与率** | 当月带来 >= 1 个转介绍注册的学员 / 当月有效学员 | 衡量转介绍活跃度 |
| **打卡率** | 当月转码且分享的学员 / 当月有效学员 | 转码 = 生成转介绍链接/二维码 |
| **带新系数** | 当月 B 学员注册人数 / 当月带来注册的 A 学员人数 | 每个推荐人平均带来几个新注册 |
| **带新注册数** | 当月 A 学员带来的 B 注册人数 | 绝对值 |
| **带货比** | 当月推荐注册人数 / 当月有效学员 | 整体池子的产出效率 |

### 漏斗指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **注册** | B 学员通过转介绍链接注册 | Leads 入口 |
| **预约率** | 预约体验课 / 注册 | 首次预约，不计多次 |
| **预约出席率** | 出席体验课 / 预约 | 首次出席，不计多次 |
| **出席付费率** | 付费 / 出席 | 体验课后愿意付费的比例 |
| **注册付费率** | 付费 / 注册 | 端到端转化率 |
| **客单价** | 付费金额(USD) / 付费单量 | 单位付费产出 |

### 效能指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **效能指数** | 付费占比 / 注册占比 | > 1.0 = 高 ROI 渠道 |
| **围场转率** | 围场内付费 / 围场内注册 | 各围场的转化效率 |

## 用户角色

| 角色 | 说明 |
|------|------|
| **UserA / A学员** | 现有学员（推荐人） |
| **UserB / B学员** | 被推荐的新学员（被推荐人） |
| **推荐人** | = UserA，带来新注册的学员 |

## 跟进相关

| 术语 | 说明 |
|------|------|
| **课前拨打** | 体验课前 CC 主动电话联系 |
| **课后拨打** | 体验课后 CC 跟进电话 |
| **有效接通** | 通话时长 >= 120 秒 |
| **拨打覆盖率** | 本月拨打学员数 / 应拨打学员数 |
| **接通覆盖率** | 本月接通学员数 / 应拨打学员数 |
| **有效接通覆盖率** | 本月有效接通学员数 / 应拨打学员数 |

## 其他术语

| 术语 | 说明 |
|------|------|
| **T-1 数据** | 今天处理的是昨天的数据 |
| **GAP / 目标缺口** | 进度落后多少（负 = 落后，正 = 超前） |
| **pp（百分点）** | 百分比的差值（如 60% → 40% = -20pp） |
| **ROI** | 投入产出比（花 1 块赚几块） |
| **MoM / 环比** | 月度环比（本月 vs 上月） |
| **YoY / 同期** | 年度同比（本月 vs 去年同月） |
| **转码** | 生成/激活转介绍专属二维码或链接 |
| **次卡** | 课时卡，按次数计费的课程套餐 |
