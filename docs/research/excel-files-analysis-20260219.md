## 核心指标定义汇总

### 1. 转化漏斗指标
```python
注册付费率 = 付费人数 / 注册人数
预约率 = 预约人数 / 注册人数
预约出席率 = 出席人数 / 预约人数
出席付费率 = 付费人数 / 出席人数
```

### 2. 转介绍效率指标
```python
转介绍参与率 = 带新参与A学员数 / 月底有效学员数
带新系数 = 总带新B学员数 / 带新参与A学员数
转介绍带货比 = 推荐注册数 / 有效学员数
打卡率 = 打卡人数 / 月底有效学员数
拨打率 = 当月有CC拨打学员数 / 月底有效学员数
触达率 = 当月有CC接通学员数 / 有效学员数
```

### 3. 业绩指标
```python
转介绍占比% = 转介绍业绩 / 新单业绩
客单价 = 转介绍业绩 / 单量
转化率 = 单量 / 注册数
```

### 4. ROI 计算模型
```python
# 次卡 ROI
次卡ROI = 实际营收 / 实际赠送次卡成本
次卡成本 = 赠送次卡数 x 次卡成本单价 (1.31 USD)

# 现金 ROI
现金ROI = 实际营收 / 实际赠送礼品&佣金成本
现金成本 = Σ(佣金支出 + 实物兑换 + 激励奖金)

# 综合 ROI
综合ROI = 实际营收 / (次卡成本 + 现金成本)
```

### 5. 目标完成率
```python
完成率 = 实际值 / 目标值
目标GAP = (实际值 - 目标值) / 目标值
环比GAP = (本月值 - 上月值) / 上月值
实际vs.预算% = (实际ROI - 目标ROI) / 目标ROI
```

## 关键发现

### 业务逻辑洞察

1. **围场效率递减规律**
   付费当月（M0）是转介绍黄金期，参与率 22.4%，随时间快速衰减至 M6+ 的 2.4%。**建议**：重点资源投入 M0-M1 用户。

2. **ROI 未达标警报**
   文件3样本数据显示 ROI 实际值（0.5/0.4）远低于目标值（40/60），**成本控制**是关键痛点。

3. **季节性波动**
   文件1历史数据显示 12月转化率异常（18.7% vs 常态 23-25%），需在年度规划中预留缓冲。

4. **窄口径 vs 宽口径**
   CC窄口径注册付费率（32%）显著高于宽口径，但宽口径注册量更大，需平衡质量与数量。

5. **CC 个人目标差异大**
   文件2显示个人月度目标跨度 6,500-18,000 USD，需根据 Aging（工作年限）分层考核。

### 数据质量问题

1. **数值单位不一致**
   - 文件1金额字段为**美元**（USD）
   - 文件2同时包含 USD 和 THB（汇率约 34）
   - 需统一单位转换逻辑

2. **时间格式多样**
   - 文件1: `202601` (YYYYMM 数字)
   - 文件1 Sheet2: `2025-01-01 00:00:00` (日期时间)
   - 文件1 Sheet3: `45658` (Excel 日期序列号)

3. **合并单元格复杂**
   文件1 "26年目标拆解" 有 31 个合并单元格区域，解析时需注意补齐逻辑。
