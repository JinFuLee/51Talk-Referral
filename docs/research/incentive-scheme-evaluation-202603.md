## 六、最终推荐

### 推荐方案: C — 带 1 人完成出席 → A 得 ฿100 优惠券

**选它的理由:**
1. **覆盖面**: 345 人能拿到（≥2 只有 24 人，差 14 倍）
2. **预算可控**: 月费用约 $1,300，占月收入的 0.6%
3. **注水风险低**: B 要坐完 25 分钟课 + ฿100 不可套现，即使 20% 注水 ROI 仍有 144 倍
4. **边际回报极高**: 比 ≥2 多花 $944 → 多赚 $115,000

**完整预期（10% 注水）:**

| 指标 | 数值 |
|------|------|
| 月度总投入 | $1,339（฿45,520） |
| 注册 User B | 1,318 人 |
| 预约 | 1,022 人 |
| 完成出席 | 670 人 |
| 付费 | 237 人 |
| **总收入** | **$226,081** |
| **ROI** | **168 倍** |
| 获客成本占客单价 | 0.62% |

**白话**: 每月花不到一万泰铢，走完注册→预约→出席→付费全漏斗后，预期产生 237 个付费学员，赚 $22.6 万。就算一成是帮忙凑数的，还是赚 168 倍。凑数的人也真坐完了课，说不定几个最后真买了。

**不推荐的方案:**
- ≥3 出席: 4 人能拿到，等于没活动
- ≥2 出席: 24 人能拿到，撬不动大盘
- 双向 A+B: 成本翻倍，增量未知，先不冒险
- 阶梯式: 最公平但执行复杂，系统改造成本高
