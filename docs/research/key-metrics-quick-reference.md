## 核心公式定义

### 转化漏斗
```python
注册付费率 = 付费人数 / 注册人数
预约率 = 预约人数 / 注册人数
预约出席率 = 出席人数 / 预约人数
出席付费率 = 付费人数 / 出席人数
```

### 转介绍效率
```python
转介绍参与率 = 带新参与A学员数 / 月底有效学员数
带新系数 = 总带新B学员数 / 带新参与A学员数
转介绍带货比 = 推荐注册数 / 有效学员数
打卡率 = 打卡人数 / 月底有效学员数
拨打率 = 当月有CC拨打学员数 / 月底有效学员数
触达率 = 当月有CC接通学员数 / 有效学员数
```

### 业绩指标
```python
转介绍占比% = 转介绍业绩 / 新单业绩
客单价 = 转介绍业绩 / 单量
转化率 = 单量 / 注册数
```

### ROI 计算
```python
次卡ROI = 实际营收 / (赠送次卡数 × 1.31 USD)
现金ROI = 实际营收 / (佣金 + 实物兑换 + 激励)
综合ROI = 实际营收 / (次卡成本 + 现金成本)
```

### 目标达成
```python
完成率 = 实际值 / 目标值
目标GAP = (实际值 - 目标值) / 目标值
环比GAP = (本月值 - 上月值) / 上月值
```

## 数据质量注意事项

1. **金额单位混用**
   - 文件1金额 = USD
   - 文件2金额 = USD + THB（汇率 34）
   - 统一时转换为 USD

2. **时间格式多样**
   - `202601` (YYYYMM 数字)
   - `2025-01-01 00:00:00` (日期时间)
   - `45658` (Excel 日期序列号)

3. **窄口径 vs 宽口径**
   - CC窄口径 = CC链接绑定（高质量，注册付费率 32%）
   - SS窄口径 = SS链接绑定（中质量，注册付费率 28%）
   - 宽口径 = UserA学员链接（低质量，但量大）

## 典型分析流程

### 1. 月度目标达成分析
```
1. 从文件1读取月度目标（26年目标拆解 Sheet）
2. 从文件2读取 CC 个人目标
3. 与实际数据对比，计算完成率
4. 按团队/个人生成排名
```

### 2. ROI 深度拆解
```
1. 从文件3读取成本单价参数
2. 统计实际赠送次卡数、佣金支出
3. 计算次卡 ROI、现金 ROI、综合 ROI
4. 与目标 ROI 对比，生成达成分析
```

### 3. 围场效率分析
```
1. 从文件1读取围场分段数据（25年转介绍数据 Sheet）
2. 按 M0/M1/M2/M3-M5/M6+ 聚合参与率、打卡率
3. 识别最优触达时间窗口
4. 生成资源分配建议
```
